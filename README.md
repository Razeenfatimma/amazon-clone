# Amazon Clone - High-Fidelity Web Interface
A front-end replica of the Amazon web interface built to master complex CSS layouts, responsive design, and UI/UX principles.

# Project Overview
This project focuses on recreating the core visual elements of an e-commerce platform. The goal was to build a clean, high-fidelity interface that remains consistent across all devices and screen sizes.

## Tech Stack
* HTML5: Semantic structuring for accessibility.
* CSS3: Custom styling using Flexbox and Grid layouts.
* Responsive Design: Media queries implemented for mobile, tablet, and desktop views.

## Key Features
* Navigation Bar: Styled search bar, account menus, and cart icon.
* Product Grids: Responsive layouts for various categories including Furniture and Electronics.
* Sign-Up/Login UI: Professional form styling with centered layouts.
* Shopping Cart: Detailed cart layout with order summary and tax logic visuals.

## Project Structure
* index.html - Main landing page
* signin.html - Account creation and login page
* cart.html - Shopping cart interface
* style.css - Main stylesheet for the entire project
* /Images - Asset folder containing project images
